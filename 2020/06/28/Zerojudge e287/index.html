Zerojudge e287

題敘

https://zerojudge.tw/ShowProblem?problemid=e287
給一張地圖，起點從權重最小的點開始，每次往四周沒走過且權重最小的點移動，求移動權重的總和

想法

用DFS照做即可

Code

複雜度

輸入時間複雜度為 $O(nm)$
DFS最多每個點都遍歷過，且每個點遍歷時間複雜度為 $O(1)$ ，DFS總時間複雜度為 $O(nm)$
總時間複雜度 $O(nm)$
