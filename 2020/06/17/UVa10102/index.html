想法

針對每個起點(1)做BFS，找到最近的3得到距離，再從其中找最大值

Code

複雜度

  • 輸入
    $O(N^2)$
  • BFS
    最糟會將地圖都跑過一遍，為$O(N^2)$
    而起點最多N個
    $O(N^3)$

整體複雜度: $O(N^3)$
